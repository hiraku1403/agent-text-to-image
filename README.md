# agent-text-to-image
Agent IA - Workflow -  Text to Image 2.0 

O Problema: Clientes e criadores de conteúdo demoram horas para criar roteiros e traduzi-los em referências visuais precisas para geração de imagens.

A Solução: Um Agente de IA Orquestrado que atua como Diretor de Arte, estruturando roteiros brutos e utilizando pipelines de tratamento de strings (Regex) via JavaScript para acionar APIs de difusão latente (FLUX via Pollinations) em tempo real.
